# artefact-store

Centralised store for NotebookLM-generated artefacts (audio overviews, video walkthroughs, infographics, slide decks) across all [NetDevAutomate](https://github.com/NetDevAutomate) repos.

**Live site:** [artefacts.netdevautomate.dev](https://artefacts.netdevautomate.dev)

## Structure

```
<repo-name>/
└── artefacts/
    ├── index.html           # Player page
    ├── audio_overview.mp3   # Audio deep-dive
    ├── video_overview.mp4   # Video walkthrough
    ├── infographic.png      # Architecture at a glance
    └── slides.pdf           # Presentation-ready deck
```

## How artefacts are published

Artefacts are generated by [notebooklm-repo-artefacts](https://github.com/NetDevAutomate/notebooklm-repo-artefacts) and published here via `repo-artefacts pipeline --store`.

Source repos contain only links to this store — no binary files.

## Adding a new repo

```bash
repo-artefacts pipeline /path/to/repo --store NetDevAutomate/artefact-store
```
